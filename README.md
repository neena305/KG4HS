# KGQA4HS

KGQA4HS is a domain-specific Knowledge Graph Question Answering (KGQA) system designed for Hindu Scriptures. It enables users to ask natural language questions and retrieves structured, explainable answers using a knowledge graph.

---

## 🔹 Features

- Natural Language Question Answering  
- Knowledge Graph-based reasoning using Neo4j  
- Tabular answer representation  
- Interactive graph visualization (Vis Network)  
- Click-based node expansion  
- Color-coded nodes based on scripture domain  
- Entity search functionality  
- Response time display (performance indicator)  

---

## 🔹 Domains Covered

- Ramayana  
- Mahabharata  
- Vedas  
- Bhagavad Gita  

---

## 🔹 Technology Stack

- **Backend:** Flask (Python)  
- **Database:** Neo4j (Cypher Query Language)  
- **Frontend:** HTML, CSS, JavaScript  
- **Visualization:** Vis Network
